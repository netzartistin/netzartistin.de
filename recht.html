Beim Besuch meiner Website werden keine personenbezogenen Daten erhoben oder gespeichert. Durch den auf Ihrem Endgerät zum Einsatz kommenden Browser werden Informationen an den Server meiner Website gesendet. Diese Informationen werden temporär in den Logfiles des Servers gespeichert. Folgende Informationen werden dabei erhoben:

  • IP‐Adresse des anfragenden Rechners

  • Name und URL der abgerufenen Datei

    • Ihres Access‐Providers
